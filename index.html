Carson-Dellosa works with various technology/educational standards organizations to promote the development of the next generation of ebooks for education. In support of these efforts, Carson-Dellosa is making a free production ebook available for use by schools teaching about epub3-based ebooks and practitioners participating in the evolution of ebook technology standards.

Supported EPUB3 Readers

The book works in Readium 0.9.1 version. So, simply extract the epub (~45MB) from "epub_demo.zip" and open in readium. A for profit version of the book with all of the epub3 and read-along features (if supported) is available in all major ebook ecosystems.

Features

The ebook includes a quiz and animation demonstrating data communication between the ebook and servers on the internet.

Animation

Submarine / Elephant animation – the roughly rendered submarine will turn when clicked and dragged or will go up and down AFTER you once click the up/down button. When you drag it up/down and release the button, the location is communicated back to a server in Moscow and the PSI is calculated and returned. If the PSI is between 50-55, then a picture of an elephant is displayed. Source code for the server application is available in the download section.

Quiz

A Quiz uses the Tincan api with data being stored in Rustici Software's SCORM cloud service. It is ten questions followed by graded results. Ability to clear, logout and restart the quiz are included. You will need to create a free account in the SCORM cloud Sign up.

You can manage the account at Sign in.
